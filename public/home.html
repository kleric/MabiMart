Welcome to MabiMart!

Introductions

Hi there! I see you found the site, cool! Anyways, things are still under construction. I began working on this site 6/4/2014, so lots of things are still a big work-in-progress. Anyways, the plan is the make a web app where it's easy to bid and auction off in game items for Mabinogi. Honestly I'm too lazy to stand around waiting to sell things with a party ad, so I figured we might as well have something like Basil Market.

I am starting completely from scratch, so I've still got a lot to write, including a user registration, the auction system itself, etc. So don't expect too much, too soon. This is mostly a fun project to learn/practice my web skills. So sorry if things slow down if I get busy with life.

If you found this from in game, I probably told you about it. Anyways, It's me Rhaenyx, hi!
